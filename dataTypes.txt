Variables : are used to store data in a program. They have a name and a type, which determines the kind of data they can hold.

Data Types 
    - Primitive Data Types : are the basic data types that are built into the Java language. They include:
        - byte : for whole numbers range from -128 to 127
        - short : for whole numbers range from -32,768 to 32,767
        - int : for whole numbers range from -2,147,483,648 to 2,147,483,647
        - long : for whole numbers range from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
        - float : for decimal numbers with single precision (up to 7 decimal places)
        - double : for decimal numbers with double precision (up to 15 decimal places)
        - char : for single characters (e.g., 'a', '1', '$')
        - boolean : for true/false values
    - Non Primitive Data Types / Reference Data Types : are data types that are not built into the Java language, but are created by the programmer. They include:
        - String : for sequences of characters (e.g., "Hello, World!")
        - Arrays : for collections of data of the same type (e.g., int[] numbers = {1, 2, 3})
        - Classes : for creating custom data types (e.g., class Person { String name; int age; })
        - Interfaces : for defining a contract that classes can implement
        - enum : for defining a set of named constants (e.g., enum Day { MONDAY, TUESDAY, WEDNESDAY })
